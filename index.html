O que é o MongoDB ?
Um Banco de Dados NoSQL, de alta performance, Sem esquemas, Orientado a documentos. Desenvolvido pela 10gen, e hoje é o maior banco de dados NoSQL do Mundo
NoSQL ?, que termo é esse ? NoSQL (not-only-sql), é um termo genérico para uma classe de Banco de Dados Não Relacionais que rompe a longa história de banco de dados relacionais com propiedades ACID ... (wikipédia)
Principais Bancos de Dados NoSQL
  • MongoDB

  • Db4o
  • Redis
  • Cassandra
  • HBase

Principais vantagens de se utilizar NoSQL
  • Escalabilidade horizontal
  • Performance
  • Sem esquemas
  • Processamento de informações com Map/Reduce
